Adds a customizable scroll to top button to your WordPress site.

== Description ==

This plugin adds a customizable scroll to top button to your WordPress site. You can easily configure the button's appearance through CSS.

== Installation ==

1. Upload the `my-scroll-to-top` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

The plugin will automatically add a scroll to top button to the footer of your site. You can further customize the button's appearance by modifying the CSS in the `codeconfig-scroll-to-top.css` file located in the plugin directory.

== Frequently Asked Questions ==

= Can I customize the appearance of the scroll to top button? =

Yes, you can customize the appearance of the button by modifying the CSS in the `codeconfig-scroll-to-top.css` file located in the plugin directory.
